# Md Sourav Oyaj - Personal Portfolio

A modern, responsive, and interactive portfolio website showcasing web and app development projects.

![Portfolio Preview](https://mr-captrox.github.io/portfolio2/images/s1.jpg)

## 🚀 Live Demo
Check out the live site here: [https://mr-captrox.github.io/portfolio2/](https://mr-captrox.github.io/portfolio2/)

## ✨ Features
- **Responsive Design**: Optimized for all devices using Tailwind CSS.
- **Dark Mode**: Toggle between light and dark themes with persistent storage.
- **Project Showcase**: Detailed cards for various projects with GitHub links.
- **Animated UI**: Interactive hero slider and scroll interactions.
- **Contact Integration**: Functional contact form powered by Formspree.

## 🛠️ Built With
- **Frontend**: HTML5, Tailwind CSS
- **Icons**: Bootstrap Icons
- **Logic**: Vanilla JavaScript
- **Backend**: Formspree (Form Handling)

## 📁 Repository Structure
- `index.html`: Main application file.
- `images/`: Visual assets and project screenshots.
- `pic2.jpg`: Primary profile picture.
